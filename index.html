ICONO

Pure css icons, with only one element

Download icono.min.css Github repo
<link rel="stylesheet" href="icono.min.css">

Also you can load icono from cdn, courtsey from KeyCDN

<link rel="stylesheet" href="https://icono-49d6.kxcdn.com/icono.min.css">

Usage:

<i class="icono-mail"></i> <div class="icono-mail"></div>
